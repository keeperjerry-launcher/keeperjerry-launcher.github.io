# Информация о сайте

На данном [сайте](https://launcher.keeperjerry.ru) размещается документация по лаунчеру. Процесс по дополнению документации полностью автоматизирован.

Предложить свои исправления для документации можно в специальную ветку `social`

## Используемые ресурсы

* [GitHub Pages](https://pages.github.com/)
* [VuePress](https://v2.vuepress.vuejs.org/)
